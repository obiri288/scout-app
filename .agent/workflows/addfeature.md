# Workflow: Add Feature (/addfeature)

## Workflow-Schritte:

1. Analysiere `code-ownership.md` und `architecture.md`.
2. Prüfe, welche shadcn/ui Komponenten bereits existieren oder hinzugefügt werden müssen.
3. Plane die Implementierung des Features unter Berücksichtigung von Framer Motion für eine flüssige UX (z.B. Enter-Animationen).
4. Präsentiere einen Implementierungsplan.
